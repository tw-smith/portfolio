PROJECTS

Analogue EXIF

Digital cameras automatically store photo metadata such as time, date and exposure settings with the image file. When practicing analogue photography, it is necessary to manually record the same information. This means having to carry a notebook and pen, which are easily forgetten. This web app allows the user to electronically record and store the same information using their mobile phone - something which most of us have with us at all times!

At it's core, this is a CRUD SPA built with Angular. The backend framework is built in Flask with SQLite used as the database solution.

Recipe Reader

When planning meals for the coming week, it can be tedious going through all the recipes and collating the various ingredients. This app aims to solve that problem by allowing the user to simply take photos of each of the recipes they want to use before creating a shopping list for the user.

The app uses the Google Cloud Vision API to perform OCR on the photo uploaded by the user before attempting to parse the ingredients list. The user is given the opportunity to correct incorrectly parsed entries. Once the user has uploaded a number of photos, the app creates a shopping list, with all common ingredients collated.

Tour Tracker

I am a big fan of cycle touring and I like to plot the routes I have followed on a map day by day so friends and family back home can follow my progress. Previously, this meant downloading individual GPS track data files before manually joining them and uploading to a Google Maps account.

This app allows the user to connect to their account on a number of popular fitness tracking services (Strava, RideWithGPS etc) and specify a date range. The app will then retrieve the GPS track data for all activities in that date range before plotting them on a map which the user can then share or embed in their own website.
